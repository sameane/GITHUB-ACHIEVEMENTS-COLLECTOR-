## 1. Quickdraw ðŸ¹
**Condition:** Close an issue or pull request within 5 minutes of opening it.

**Steps:**
1. Go to any repository you own.
2. Click on the **Issues** tab.
3. Create a **New Issue** with any title/description.
4. Immediately after creating it, click **Close Issue** (ensure this is done in less than 5 minutes).

## 2. Pull Shark ðŸ¦ˆ
**Condition:** Open and merge pull requests (PRs) into your repositories.

**Steps:**
1. Open any file in your repository and click the **Edit (pencil icon)**.
2. Make a small change (e.g., add a space).
3. Under the "Commit changes" section, select **"Create a new branch for this commit and start a pull request"**.
4. Click **Propose changes**.
5. Click **Create pull request**.
6. Finally, click **Merge pull request** and then **Confirm merge**.
*(Repeat this multiple times to level up the badge).*

**Condition:** Be a co-author on a merged commit.

**Steps:**
1. You need a second GitHub account (or a friendâ€™s account).
2. Add the second account as a **Collaborator** in your repository (Settings -> Collaborators).
3. Use **GitHub Desktop** to make a commit.
4. Before clicking "Commit to main", click the small **"Co-authors"** icon at the bottom of the commit summary.
5. Search for and select the second account.
6. Commit and Push the changes.
7. Create a PR and Merge it.

## 5. Galaxy Brain ðŸ§ 
**Condition:** Have your answer accepted in a "GitHub Discussions" Q&A.

**Steps:**
1. Visit the [GitHub Community Discussions](https://github.com/orgs/community/discussions).
2. Look for questions in the **Q&A** category and provide a helpful, honest answer.
3. If the person who asked the question marks your response as the **"Accepted Answer"** at least twice, you will receive the badge.
